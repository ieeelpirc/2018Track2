Requirement: Ubuntu 16.04
The steps to install Caffe2 is as follows:
1. Install dependencies
2. Install Cuda8.0
3. Install cuDNN8.0
4. Install Caffe2 from Source

Note that if you have already installed Cuda8.0 or cuDNN8.0, please skip these steps.
You can run "test.sh" to test the installation of Caffe2. If the output is "success", your installation is correct.
